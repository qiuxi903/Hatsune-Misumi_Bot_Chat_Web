**1. OneBot v11消息格式问题**

问题：发送string格式消息时，AstrBot返回错误：
```
aiocqhttp: 无法识别的消息类型: 你好，此条消息将被忽略。如果您在使用 go-cqhttp，请将其配置文件中的 message.post-format 更改为 array。
```

原因：aiocqhttp库要求消息格式为array（列表），而不是string（字符串）。

解决方案：
```javascript
// 错误格式
message: "你好"

// 正确格式
message: [{ type: 'text', data: { text: '你好' } }]
```
